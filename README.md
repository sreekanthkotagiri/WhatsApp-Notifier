# WhatsAppAutoNotifier

## Description
WhatsAppAutoNotifier is a NestJS application designed to automate notifications via WhatsApp. This project serves as a skeleton for building a notification system that can be extended with additional features.

## Installation

To get started with the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd WhatsAppAutoNotifier
   ```

3. Install the dependencies:
   ```
   npm install
   ```

## Running the Application

To run the application, use the following command:
```
npm run start
```

The application will start on the default port (3000). You can access it at `http://localhost:3000`.

## Testing

To run the end-to-end tests, use the following command:
```
npm run test:e2e
```
